## React Portfolio

## User Story
* As a developer this week's challenge is to make a Portfolio with React. It is to showcase the technologies which learnt in the bootcamp. Same is highligthed in the resume and the projects.

* It is also useful for applyng jobs and collaborating with other developers.

## Accpetance Criteria
* It's done when the single page application is loaded. Its presented with a home page with a header section and a footer section.
* It's done when the header is presented with a logo and mentioning different navigation pages like About Me, Resume, Projects and Contact.
* It's done when any of the link is clicked the url changes and it takes to respective page and the logo redirects to home page.
* It's done when the About me is presented with a photo and and a short bio.
* It's done when the Resume is presented with the technologies learnt and a downloadable link.
* It's done when the projects page is showcased with 6 projects with the name, picture, a small description, github links and deployment or walkthrough video links.
* It's done when the Contact Me page is presented with a form with fields for a name, email address and a message section. The notification comes when any of the field are empty or the email is not properly formatted.
* It's done when the footer is presented with the github and linkedIn logo and links.
